# Chic Boutique — Local Business Website

A modern, responsive website designed for a fashion boutique.  
This project focuses on clean design, strong visual identity, and a smooth user experience across all devices.

---

## ✨ Project Overview

Chic Boutique is a frontend project that simulates a real-world website for a small fashion business.  
The goal was to create a visually elegant, mobile-first layout that reflects a luxury yet approachable brand.

The website includes all essential sections a local business needs:
- Hero introduction
- About section
- Seasonal collections
- Contact form
- Footer with branding

---

## 🎯 Goals & Design Decisions

- **Modern & clean aesthetic**  
  Neutral tones, warm gold accents, and elegant typography were chosen to reflect a boutique brand identity.

- **Mobile-first approach**  
  The layout was built with responsiveness in mind, ensuring a smooth experience on mobile, tablet, and desktop.

- **Minimal JavaScript**  
  JavaScript is used only where it improves usability (mobile navigation and form feedback).

- **Scalable structure**  
  CSS variables and reusable components make future updates easy.

---

## 🛠️ Technologies Used

- **HTML5** — semantic structure and accessibility
- **CSS3** — Flexbox, Grid, custom properties, responsive design
- **JavaScript (ES6)** — basic interactivity and UX enhancements
- **Git & GitHub** — version control and project history

---

## 📱 Key Features

- Responsive navigation with mobile menu toggle
- Hero section with high-quality imagery
- Collection cards with consistent layout
- Accessible contact form with user feedback
- Clean, readable typography and spacing
- Organized and maintainable codebase
